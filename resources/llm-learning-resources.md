## 📖 教程类 (Tutorials)

### 1. LLM-Universe (Datawhale) ⭐ 6.8K
- **描述**: 面向小白开发者的大模型应用开发教程，从0到1系统化学习
- **亮点**: 实践导向，构建"个人知识库助手"项目，覆盖Prompt Engineering、RAG等核心技术
- **GitHub**: https://github.com/datawhalechina/llm-universe
- **在线阅读**: https://datawhalechina.github.io/llm-universe/
- **适合人群**: 完全零基础，想快速上手LLM应用开发

### 2. LLM-Course (mlabonne) ⭐ 4.6K
- **描述**: 系统化的大型语言模型学习课程，从入门到进阶
- **亮点**: 模块化设计，理论与实践结合，涵盖量化、微调、模型合并等
- **GitHub**: https://github.com/mlabonne/llm-course
- **中文版**: https://github.com/yuanzhongqiao/llm-course-cn
- **适合人群**: 有一定编程基础，想深入理解LLM原理

### 3. Hands-On Large Language Models ⭐ 16.7K
- **描述**: Jay Alammar和Maarten Grootendorst合著的官方配套代码仓库
- **亮点**: 被誉为"图解LLM"，可视化讲解，实战代码丰富
- **GitHub**: https://github.com/HandsOnLLM/Hands-On-Large-Language-Models
- **适合人群**: 喜欢图文并茂学习方式的开发者

### 4. Microsoft AI Agents for Beginners ⭐ 5.2K
- **描述**: 微软官方AI智能体入门课程
- **亮点**: 12周课程，涵盖AI基础理论、深度学习、NLP、Transformer等
- **GitHub**: https://github.com/microsoft/ai-agents-for-beginners
- **适合人群**: 从机器学习过渡到AI的开发者

### 5. Microsoft ML-For-Beginners
- **描述**: 微软官方机器学习入门计划，为期12周
- **亮点**: 监督学习、无监督学习、分类、回归、聚类、时间序列分析等基础知识
- **GitHub**: https://github.com/microsoft/ML-For-Beginners
- **适合人群**: 机器学习零基础

### 6. PyTorch到大语言模型学习路线
- **描述**: 程序员学习LLM的完整路线图，包含各章节笔记
- **亮点**: 从PyTorch基础到LLM应用，循序渐进
- **适合人群**: 有编程基础，想从PyTorch开始学习LLM

### 7. 动手学Ollama (Datawhale)
- **描述**: CPU玩转大模型部署，零成本本地运行LLM
- **亮点**: 在线阅读，实操性强
- **在线阅读**: https://datawhalechina.github.io/handy-ollama/
- **适合人群**: 想在本地部署LLM的开发者

## 🛠️ 工具类 (Tools)

### 1. LangChain ⭐ 最流行
- **描述**: LLM应用开发框架，提供完整的工具链
- **GitHub**: https://github.com/langchain-ai/langchain
- **亮点**: 链式调用、Agent、记忆、RAG等一站式解决方案
- **适用场景**: 构建复杂的LLM应用，如聊天机器人、问答系统等
- **快速开始**:
  ```bash
  pip install langchain
  ```

### 2. LlamaIndex ⭐ RAG首选
- **描述**: 专注数据索引和检索的框架，构建基于LLM的数据驱动应用
- **GitHub**: https://github.com/run-llama/llama_index
- **亮点**: 最流行的RAG框架之一，企业级应用标准选择
- **适用场景**: 构建基于私有数据的问答系统

### 3. Dify ⭐ 开源LLM应用开发平台
- **描述**: 开源的LLM应用开发平台，直观界面
- **GitHub**: https://github.com/langgenius/dify
- **亮点**: AI工作流、Prompt IDE、RAG、Agent、后端即服务
- **适用场景**: 快速从原型到生产

### 4. MaxKB ⭐ 企业级智能体平台
- **描述**: 强大易用的开源企业级智能体平台
- **GitHub**: https://github.com/1Panel-dev/MaxKB
- **亮点**: 支持多种模型，集成RAG、Agent、MCP Server等
- **适用场景**: 企业级AI应用部署

### 5. Langfuse ⭐ LLM工程平台
- **描述**: 开源LLM工程平台，专为开发人员设计
- **GitHub**: https://github.com/langfuse/langfuse
- **亮点**: 提示管理、评估、测试游乐场，与LangChain、LlamaIndex集成
- **适用场景**: LLM应用的监控、调试、优化

### 6. Ollama ⭐ 本地LLM运行
- **描述**: 本地运行大语言模型的工具，支持CPU推理
- **GitHub**: https://github.com/ollama/ollama
- **亮点**: 零成本运行，支持多种开源模型
- **适用场景**: 本地开发、隐私保护、成本控制

### 7. ChromaDB / Pinecone (向量数据库)
- **描述**: 用于存储和检索Embedding向量
- **ChromaDB**: https://github.com/chroma-core/chroma
- **Pinecone**: https://www.pinecone.io/
- **适用场景**: RAG系统的核心组件

## 📄 论文实现类 (Paper Implementations)

### 1. Hugging Face Transformers ⭐ 最流行
- **描述**: 预训练模型的统一框架
- **GitHub**: https://github.com/huggingface/transformers
- **亮点**: 支持几乎所有主流LLM模型，开箱即用
- **适用场景**: 模型推理、微调、研究

### 2. Meta Llama Cookbook
- **描述**: Meta官方Llama模型使用指南和示例
- **GitHub**: https://github.com/meta-llama/llama-cookbook
- **亮点**: 官方最佳实践，包含微调、部署等

### 3. Annotated Transformer
- **描述**: Transformer论文的代码实现，带详细注释
- **GitHub**: 搜索 "annotated transformer"
- **适用场景**: 深入理解Transformer架构

### 4. NanoGPT
- **描述**: 最简化的GPT实现，适合学习原理
- **GitHub**: https://github.com/karpathy/nanoGPT
- **亮点**: Andrej Karpathy出品，代码简洁易懂
- **适用场景**: 理解GPT训练和推理原理

### 5. GPT-2 from Scratch
- **描述**: 从零实现GPT-2
- **GitHub**: 搜索 "gpt2 from scratch"
- **适用场景**: 深入理解LLM架构

## 🎯 学习路径建议

### 第1阶段: 入门（1-2周）
1. 完成 LLM-Universe 教程
2. 学习 LangChain 基础
3. 跑通第一个API调用

### 第2阶段: 进阶（3-4周）
1. 学习 LlamaIndex 实现RAG
2. 了解 LLM 原理（LLM-Course）
3. 尝试本地部署（Ollama）

### 第3阶段: 实战（5-8周）
1. 构建完整项目（知识库助手、聊天机器人等）
2. 学习微调技术
3. 部署到生产环境

### 第4阶段: 深入（9-12周）
1. 阅读论文，理解前沿技术
2. 参与开源项目贡献
3. 探索Agent、多模态等高级主题
