## 🚀 实战项目

### 📚 名著改写器 (Classic Literature Rewriter)

一个基于 LLM 的经典文学风格改写系统，让用户可以将经典名著改写成不同风格（诙谐、严肃、活泼等），保留章节结构，深度润色。

**功能特点：**
- 8 种改写风格（诙谐幽默、严肃庄重、活泼生动、现代口语、简洁明快、戏剧化、儿童友好、学术风格）
- 章节结构保留
- 网页界面 + API 集成（通义千问）

**项目地址：** [classic-rewriter/](classic-rewriter/)

**技术栈：**
- 后端：Flask + 通义千问 API
- 前端：HTML/CSS/JavaScript
- Prompt 工程：8 种风格模板

> 这是 LLM 应用开发的实战项目，从 Day 1 的基础直接跳到实际项目，学以致用！
